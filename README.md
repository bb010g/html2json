# html2json

Converts HTML losslessly into JSON for easier processing fast, thanks to
[html5ever][] and [serde][].

Featuring more `transmute` than you're comfortable with!

[html5ever]: https://github.com/servo/html5ever
[serde]: https://serde.rs/
